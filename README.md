## How to use

Install the project:

```bash
$ git clone https://github.com/LuccaWang1/cherry-cloud
$ virtualenv env
$ source env/bin/activate
(env) $ pip3 install -r requirements.txt
(env) $ createdb cherry-cloud
$ python3 model.py
```

To run the server:

```bash
$ python3 server.py
open localhost: https://127.0.0.1:5000/
```
